Port Proxy Service

This service proxies HTTP requests to localhost ports using the port number in the URL path.

How to use:

Access http://this-server/[port]/path to proxy to http://localhost:[port]/path

Port Range:

Valid ports: 1027 - 19999

Examples:

Web application on port 3000:

GET /3000/ → http://localhost:3000/

API endpoint on port 8080:

GET /8080/api/v1/users → http://localhost:8080/api/v1/users

WebSocket connection:

WS /4000/socket → ws://localhost:4000/socket

Features:

  • ✅ HTTP/HTTPS proxying
  • ✅ WebSocket support
  • ✅ Query parameters preserved
  • ✅ Request headers forwarded
  • ✅ Automatic path cleaning
